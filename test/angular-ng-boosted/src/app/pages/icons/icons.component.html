Icons integration

This page shows some examples of how to integrate some Solaris icons inside your project.

Find icons

You can use the Solaris Icons Finder to easily find your icons by keywords (in English or French), style, category or popularity.

Use icons in your web page

You can refer to the develop documentation to see in details the different integration possibilities: SVG sprite, web font, inline SVG, SVG as an external image or SVG in CSS background.

For the moment, the preferred technique is by using a SVG sprite file. You can generate this file directly from the Solaris Icons Finder tool.

You can also refer to Boosted icons documentation to know more about SVG sprite icons integration with Boosted.
