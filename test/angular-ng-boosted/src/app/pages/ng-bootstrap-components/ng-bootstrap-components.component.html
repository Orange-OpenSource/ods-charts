ng-bootstrap specific components

This page shows some ng-bootstrap dynamic components which are not implemented in Boosted: Datepicker, Pagination and Timepicker.

We provide a basic Orange design through a custom SCSS file, that you can import in your project and use as a basis to start.

Keep in mind that, in an eco-design approach, you should only import ng-bootstrap components that you need (see ng-bootstrap documentation).

Please take a look at this project's sources and README.md to know more about ng-bootstrap integration in an Angular project.
