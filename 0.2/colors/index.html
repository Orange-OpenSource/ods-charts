ODS Charts colors sets

ODS and OUDS colors sets are provided at the same level at the moment.

OUDS colors sets are kind of experimental at the moment, please see more in our associated Data display guidelines page. Please note that the OUDS colors are supposed to be used even in ODS project (except for ODS Functional and Highlight) right now according to the DSM guidelines page. Here is a summary:

  • OUDS Sequential colors sets are almost for the same usage as the ODS ones. They should be used when data values can be ordered from low to high. Choose accessible colors compared to the background whenever it's possible.
  • OUDS Categorical colors set is almost for the same usage as the ODS one. It should be used to distinguish discrete categories of data that are unrelated. Know that the sequence is carefully curated to maximise contrast between neighbouring colors. So the colors inside should be used in the sequencing order.
  • OUDS Functional colors set is almost for the same usage as the ODS one. It should be used to establish color association for statuses. This set should be used to reinforce meaning through data display visualization.
  • OUDS Highlight colors set is new compared to the ODS ones. It's a part of the previous ODS Functional. It should be used to draw attention on a specific data within a dataset.
  • OUDS Single colors set is new compared to the ODS ones. It's a set made to help people choosing an accessible color for single color charts. Their accessibility was tested against light (#ffffff) values and dark (#141414 and so #000000) values.
